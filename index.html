03 — การปัดเลข

กฎการปัดเลข (Rounding Rules)

เงื่อนไขการกระทำตัวอย่าง
ตัวเลขที่ปัด < 5 ปัดลง — ตัดทิ้ง 3.74 → 3.7
ตัวเลขที่ปัด > 5 ปัดขึ้น — เพิ่ม 1 3.76 → 3.8
ตัวเลขที่ปัด = 5 มีเลขตามหลัง ปัดขึ้น — เพิ่ม 1 3.751 → 3.8
ตัวเลขที่ปัด = 5 เท่านั้น (Banker's) ปัดให้ตัวก่อนหน้าเป็นเลขคู่ 2.45 → 2.4  /  2.35 → 2.4

ตัวอย่าง: ปัดให้ได้ 3 นัยสำคัญ

6.3482 → 6.35
    ↑ ตัวที่ 4 = 8 > 5 ปัดขึ้น
0.07436 → 0.0744
      ↑ ตัวที่ 4 = 6 > 5 ปัดขึ้น
9.84500 → 9.84
    ↑ ตัวที่ 4 = 5 เท่านั้น, 4 คู่ คงไว้

ลำดับขั้นตอน

1. ระบุจำนวนนัยสำคัญที่ต้องการ
2. นับตัวเลขจากซ้าย (ข้ามศูนย์นำหน้า)
3. ดูตัวถัดไปจากตำแหน่งที่ต้องการ
4. ปัดตามกฎ (<5 ตัดทิ้ง, >5 ปัดขึ้น)
5. ตรวจสอบนัยสำคัญอีกครั้ง
04 — กราฟและความชัน

กราฟเส้นตรงและความชัน (Slope)

กราฟ y = mx + c (ปรับค่าได้)

[ กราฟ y = mx + c — ดูได้ในเวอร์ชันออนไลน์ ]
ความชัน
m = Δy / Δx
จากจุดสองจุด
m = (y₂−y₁)/(x₂−x₁)
สมการเส้นตรง
y = mx + c

ขั้นตอนหาความชันจากกราฟ

01
เลือก จุดสองจุดบนเส้น ที่ห่างกันมากและอ่านค่าได้ชัดเจน
เช่น (1, 3) และ (5, 11)
02
คำนวณ Δy = y₂ − y₁ และ Δx = x₂ − x₁
Δy = 11 − 3 = 8  |  Δx = 5 − 1 = 4
03
คำนวณ m = Δy ÷ Δx แล้วปัดตามนัยสำคัญ
m = 8 ÷ 4 = 2.0
04
ตีความ: ความชัน บวก = เส้นพุ่งขึ้น  |  ลบ = เส้นลาดลง  |  0 = เส้นแนวนอน
จุดตัดแกน Y
x = 0 → y = c
แทน x = 0 ในสมการ → ได้ค่า y ที่ตัดแกน Y
จุดตัดแกน X
y = 0 → x = −c/m
แทน y = 0 แล้วแก้สมการ → ได้ค่า x ที่ตัดแกน X
ความชันกับนัยสำคัญ ผลลัพธ์ m ต้องมีนัยสำคัญเท่ากับที่น้อยสุดในการหาร เช่น Δy = 8.4 (2 นัยสำคัญ), Δx = 2.10 (3 นัยสำคัญ) → m = 4.0 (2 นัยสำคัญ)
05 — ค่าความคลาดเคลื่อน

การหาความคลาดเคลื่อนของผลลัพธ์ (Error Propagation)

กำหนด A ± a และ B ± b โดย A, B คือค่าที่วัดได้ และ a, b คือความคลาดเคลื่อน

ผลบวก
(A ± a) + (B ± b) = (A + B) ± (a + b)
ความคลาดเคลื่อนบวกกันเสมอ
(5.2 ± 0.1) + (3.4 ± 0.2)
= (5.2 + 3.4) ± (0.1 + 0.2)
= 8.6 ± 0.3
ผลลบ
(A ± a) − (B ± b) = (A − B) ± (a + b)
ลบค่า แต่ความคลาดเคลื่อนบวกกันเสมอ ไม่ลบ
(7.5 ± 0.2) − (3.1 ± 0.1)
= (7.5 − 3.1) ± (0.2 + 0.1)
= 4.4 ± 0.3
ผลคูณ
(A ± a)(B ± b) = AB ± (Ab + Ba)
กระจายแล้วตัดพจน์ ab ทิ้ง เพราะ a, b มีค่าน้อยมาก
(4.0 ± 0.2)(2.0 ± 0.1)
Ab + Ba = (4.0)(0.1) + (2.0)(0.2) = 0.4 + 0.4
= 8.0 ± 0.8
ผลหาร
A ± aB ± b = AB ± ( aA + bB ) ( AB )
นำ (a/A + b/B) คูณกับผลหาร A/B เพื่อได้ความคลาดเคลื่อน
(6.0 ± 0.3) ÷ (2.0 ± 0.2)
A/B = 6.0/2.0 = 3.0
a/A + b/B = 0.3/6.0 + 0.2/2.0 = 0.05 + 0.10 = 0.15
ความคลาดเคลื่อน = 0.15 × 3.0 = 0.45 ≈ 0.5
= 3.0 ± 0.5

06 — ตารางสรุปรวม

ตารางสรุป

การคำนวณ กฎนัยสำคัญ สูตรความคลาดเคลื่อน ตัวอย่างผลลัพธ์
+ บวก ตำแหน่งทศนิยมน้อยสุด (A+B) ± (a+b) 8.6 ± 0.3
− ลบ ตำแหน่งทศนิยมน้อยสุด (A−B) ± (a+b) 4.4 ± 0.3
× คูณ จำนวนนัยสำคัญน้อยสุด AB ± (Ab+Ba) 8.0 ± 0.8
÷ หาร จำนวนนัยสำคัญน้อยสุด A/B ± (a/A+b/B)(A/B) 3.0 ± 0.5
💡 เทคนิคจำง่าย บวก/ลบ → ดูตำแหน่งทศนิยม  |  คูณ/หาร → นับจำนวนนัยสำคัญ  |  ความคลาดเคลื่อน บวก/ลบ → สัมบูรณ์  |  คูณ/หาร → สัมพัทธ์

14. y = −3x + 6 ตัดแกน X และแกน Y ที่จุดใด?▸
ตัดแกน Y ที่ (0, 6) — แทน x=0 | ตัดแกน X ที่ (2, 0) — แทน y=0 แล้วแก้สมการ
15. Δy = 8.4 และ Δx = 2.10 ความชันมีกี่นัยสำคัญ?▸

หมวด 6 — ความคลาดเคลื่อน

16. (12.0 ± 0.5) + (3.5 ± 0.2) = ?▸

17. (5.0 ± 0.1) × (4.0 ± 0.2) = ?▸

18. (9.0 ± 0.3) ÷ (3.0 ± 0.3) = ?▸
